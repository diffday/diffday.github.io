高效团队和亚马逊经验

发表于 2021-09-14 更新于 2022-08-22 分类于 网文精摘 本文字数： 5.3k 阅读时长 ≈ 5 分钟

文章来源于infoq的摘取 https://www.infoq.cn/article/MYEKmcClGEXvKRQ7LWNY

业务模式不是孤立存在的，企业要想基业长青，不仅要有好的业务模式，还要有正确的人，永不熄火的创新引擎，既快又好的决策机制，及强有力的组织文化。

企业成败关键在人，人是根本，战略都是人想出来的，事情都是人做出来的，始于老板，终于组织文化。

你的人就是你你的企业，用什么样的人，你的企业就会变成什么样子

积极上进的工程师能产生数倍于“普通工人”的价值，二者的区别就是前者能主动发现问题、而后者只会按指令办事

高效团队

  • 软件开发迭代周期和频率越来越快
  • 管理问题
    • 团队越来越大，分布在不同区域
    • 执行力问题，流动问题，成长问题

不是所有公司都需要高效团队，但当处理的业务复杂度，交互复杂度，或服务用户数越来越多，就需要高效团队了。要又快又好，还要降低成本，如上提及的管理问题也在凸显。

如何搞成高效团队

  • 搞敏捷
  • 用权威，干不完不用下班
  • 使命感价值观，要不停地努力，这是对你好
  • KPI，996，钉钉 等等

如上保证不了你的公司你的团队去到更高的位置，解决更复杂更难的问题，**有更有价值的产出，终极的问题还是在团队的人上，再看团队中的事，再之团队领导力**。

女性婚恋相亲窘境

发表于 2021-08-13 分类于 网文精摘 本文字数： 949 阅读时长 ≈ 1 分钟

“割韭菜”的相亲平台

标题党，但又折射出相亲的矛盾。相亲的本质就是一帮最不会给女人提供情绪价值的男人遇见的了一帮最需要情绪价值的女人，再真实一点，就是一帮最不会追女人的男人遇到了一帮最难追的女人，再再真实一点，就是一帮要求很高的男人遇到了一帮要求也很高的女人，他们能不互相嫌弃吗？

现实世界满足不了的幻想，相亲平台来满足你。现实中找不到理想的另一半，相亲平台都有。实际上稍微有点吸引力的人，身边的追求者都应付不过来，哪有时间精力来相亲平台登记呢？

在婚恋市场上，一旦你进步了相亲这个环节，虽然表面看上去你有很多的选择权，但实际意味着，你其实没有选择权了。

男女方的认知偏差

男人想的是我越相亲我越不想付出，但是女人的想法恰恰相反，我越相亲我越要找一个能对我付出的，否则我等了35年白等了。反正沉默成本这么高，大不了再等三年呗，但是再三年，她的要求就更高了。但男人想的是，又过了三年，价格该打折了。所以这些男人和女人之间的需求，完全是错配的，且女性更难一些。

女性的特质囧境分析

单身人士如果相亲不能遇到真爱，怎么才能遇到真爱呢，其实这个问题关键因素有两个。

不知道对方什么需求

一是不知道对方什么需求，比如很多女生以为男人喜欢的特点是女生读世界名著，钢琴，瑜伽，对小动物有爱心，环保做义工，有份好工作，吃有机食品不吃烤串，其实这些男人都不在乎。男生在乎的无非是温柔体贴加好看。

如果把相亲的人比做滞销产品，还没找到另一半真正的原因，是你自认为的优点，根本不符合客户的需求。如果一个产品卖不出去，滞销了怎么办，这个时候，你不应该埋头生产更多不符合市场需求的产品，各种客户不需要的功能都加上去，越卖越贵。比如弹个钢琴，做个瑜伽，读书旅游，然后强行推销。而是应该立即停产，要么降价大促销，要么调整配方再出售。相亲也是一样的道理。

高估自己

其次，人都会高估自己。比如女生会高估自己的容貌，直男行规是平方/10。比如有位女性宣称自己有8分，直男心算6.4分，女生宣称自己6分中等美，实际上是3.6分。1957年美国一个实验也能证实，你觉得你的智商是否超过了社会平均？91%的美国人选择了是，反映到相亲市场也是一样的，90%的人认为自己在平均线以上，所以要求对方也要在平均线以上。

结语

所以了解对方的需求，合理评估自己，才能找到真爱。

扒家底看大佬发展

发表于 2021-06-16 更新于 2023-03-14 分类于 学而思 ， 反刍整理 本文字数： 7.2k 阅读时长 ≈ 7 分钟

出身草根的大佬占比并不高，就像2000年来泥腿子当皇帝的就两个，汉与明。两个都还出在特定的知识分子庙堂无出路的年代。除了卵巢彩票，一个普通人最大的机遇在于婚姻，上大学提供的跃迁其实都不及婚姻大。

骆驼成功穿越沙漠，成为野外生存专家，讲了很多技能，又强调说坚持和努力更重要（这些都对），就是不说自己有驼峰，所有的成功的秘诀都是很难复制的，也不会外传

扒了一下企业界大佬们的情况，发现祖传的天分，几代人的经验积累，耳濡目染的传承（素质、资源或成长条件），远超寻常农民子弟的成功几率，能更早更充分具有超越一般人的从商的意愿、见识，办事的能力。

王石年轻时参军，上大学，找工作和创业，都离不开家里的背景；柳传志的父亲在他早期或者联想早期发展过程中都起了不小的作用；

周鸿祎王兴都是中国第一批用上电脑和互联网的人，雷军可从村办小学转到县城名校，张一鸣从小听父母讨论各种科研，发明；马化腾14岁就可以买天文望远镜；冯仑从小就能看到图书馆大量的书，包括“仅供内幕批判”的读物，都离不开家庭条件的支撑。

更何况，每天饭桌上的交谈，待人接物的言传身教，不同家庭不啻天壤之别。精英家庭讨论国家政策对家庭的什么利好时，农村家庭可能又在讨论家里的老母猪又生了一窝，孩子每天要早起一小时多打一篮猪草回来。

当然事情并不绝对，李彦宏，俞敏洪，张小龙，王传福等人的成功，也证实，天道不绝，终留一线。
